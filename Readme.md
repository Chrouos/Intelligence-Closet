## 操作步驟
1. 建立資料庫 "intelligence_closet"
2. 建立資料表，請在該資料庫的環境中執行　"在SQL這個資料夾中的 -> total.sql"
3. 在 "DAO這個資料夾中 -> crudAccount.py" 裡面換成自己的帳號密碼(請注意往後這個檔案不要上傳到git)
